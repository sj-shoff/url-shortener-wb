# URL Shortener

Простой сервис для создания коротких ссылок и просмотра аналитики.

## Что умеет сервис

- Создавать короткие ссылки из длинных URL
- Показывать статистику переходов по каждой ссылке
- Работать с кастомными алиасами (короткими именами)
- Сохранять данные о каждом переходе (время, браузер, IP)

## Технические детали

- **Язык:** Go
- **База данных:** PostgreSQL
- **Кэш:** Redis

## Как запустить

### Создание .env

```bash
cp .env.example .env
```
Отредактируйте настройки в .env

### Запуск в Docker
```bash
# Запустить все сервисы
make docker-up

# Установить миграции базы данных
make migrate-up

# Остановить все сервисы
make docker-down
```

### Локальный запуск

```bash
# Запустить локально
make run
```

После запуска сервис будет доступен по адресу: `http://localhost:8002`

## Как использовать

### Через веб-интерфейс

1. **Откройте** `http://localhost:8002`
2. **Вставьте длинный URL** в поле "Оригинальный URL"
3. **Укажите алиас** (опционально) - короткое имя для ссылки
4. **Нажмите** "Создать короткую ссылку"
5. **Используйте** полученную короткую ссылку

### Просмотр статистики

1. **Перейдите** на вкладку "Аналитика"
2. **Введите алиас** вашей короткой ссылки
3. **Нажмите** "Загрузить"
4. **Смотрите** статистику переходов

## Формат алиасов

- Только английские буквы (A-Z, a-z) и цифры (0-9)
- Длина от 3 до 20 символов
- Регистр не имеет значения

## Примеры

**Создание ссылки:**
- Вход: `https://example.com/very/long/url/path/that/needs/shortening`
- Алиас: `mylink` (необязательно)
- Результат: `http://localhost:8002/s/mylink`

**Просмотр статистики для алиаса `mylink`:**
- Всего переходов: 150
- Сегодня: 25
- Этот месяц: 55
